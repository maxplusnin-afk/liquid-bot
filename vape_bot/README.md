# 🍼 Telegram Bot для каталога жидкостей

Бот для просмотра каталога электронных жидкостей с админ-панелью.

## 📋 Функционал

### Для пользователей:
- 👋 Приветствие при старте
- 🍼 Просмотр каталога жидкостей
- 👃 Детальная информация о каждой жидкости (вкус, крепость, объем)
- 📞 Контакт продавца для покупки (@maxon2205)

### Для администраторов:
- 🔐 Защищенный вход по команде /admin
- 📦 Добавление новых жидкостей (название, вкус, крепость, объем, фото)
- 📋 Просмотр всех жидкостей
- 🗑 Удаление жидкостей из каталога

## 🛠 Технологии
- Python 3.9+
- Aiogram 3.x
- SQLite
- Python-dotenv
